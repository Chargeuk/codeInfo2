## Canonical Plan Snapshot

### Description

- Story 54 speeds up ingest for very large prose-heavy files without changing the existing ingest UI or ingest REST entrypoints.
- The story adds three coordinated changes:
  - large-text chunk routing for large `.md`, `.mdx`, and `.txt` files;
  - provider-aware embedding dispatch with bounded batching and concurrency;
  - conservative AST delta behavior that skips AST only for non-AST-only deltas and otherwise reuses the existing full rebuild path.

### Acceptance Criteria

- The plan currently defines 33 acceptance criteria covering discovery metadata, chunk routing, token safety, dispatch configuration, queueing, cancellation, AST delta mode, compatibility, runtime-surface preservation, docs, and full validation.

### Out Of Scope

- No new ingest UI surface, no new ingest route, no separate worker service, no rename detector, no partial AST updates, no benchmark SLA, and no broader non-ingest performance work.
