# Apply the agent-native review settlement

Read the complete-pass settlement recommendation, every referenced batch and finding, current Git state, and the bounded plan.

If the pass was genuinely clean, record the no-work closeout without adding a task. If direct fixes occurred without task-required findings, add or update one final testing/revalidation task after all existing work. If findings require tasks, create one or more appropriately scoped implementation tasks, improve their dependencies, exit criteria, subtasks, automated proof, manual guidance, and finding provenance, then add one final testing/revalidation task after them.

Use the repository plan helpers repeatedly to check the work. Keep task ordering and current-task handoffs valid. Update the plan continuously and make the normal implementation loop able to select the new work. Do not implement task-required findings or start another review in this step.
